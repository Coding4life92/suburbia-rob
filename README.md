# Suburbia Skateboards

An interactive skateboard website featuring a real-time 3D skateboard customizer. Users can explore decks and personalize their setup with a smooth WebGL experience.

Built with modern web technologies focusing on performance, component architecture, and immersive UI.

## Tech Stack
* Framework: Next.js
* Language: TypeScript
* CMS: Prismic
* 3D Rendering: Three.js
* React 3D Renderer: React Three Fiber
* 3D Utilities: Drei
* Styling: Tailwind CSS
